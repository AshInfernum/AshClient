# Ash Client 🔥 | Peak of Anomaly Analysis

Ash Client is a Meteor Client addon designed for base hunting on DonutSMP, minimizing human effort by automatically analyzing most things in an optimized way.

Here is the Turkish Language Page -> https://github.com/AshInfernum/AshClient/blob/main/TR.md

---

## 🔍 Ash Chunk Finder

**This module is powerful enough to work like a standalone client.** Chunk Finder is a combination of multiple advanced modules working together.

It analyzes chunks using multiple ESP types (including ESP types that are exclusive to this client) and combines these results to determine whether a chunk contains a base or not.

Additionally, there is a background system that checks whether a base is “exposed” or not. All ESP systems are merged into a single module, removing the need for manual checking and making the entire process fully automated.

---

## 💎 Support Modules

While Chunk Finder detects the source, support modules handle extraction and safety operations:

- 🎂 Cake & 💎 BUD ESP: Uses Bud and Cake analysis to detect possible bedrock spawner bases and reveal hidden base locations.

- 🛡️ Player Detect: Detects nearby players and sends notifications via Discord webhook. With Auto Quit, if a player enters your area while you have valuable items, it automatically disconnects you from the game.

- 💰 FakePay: A module that simulates fake payments in DonutSMP format, making it look like real transactions.

- 🚨 Totem Alert: Plays an anvil sound when you forget to equip a totem in your off-hand, helping prevent unexpected deaths.

- 🚩 Coords Snapper: Sends your current coordinates to a webhook, useful for tracking locations discreetly.

## 🚀 Getting Started

Launch Ash Client and go to the Ash category.

Open the auth module and copy your HWID.

Send your HWID to the administrator and wait to be whitelisted.

After approximately 5 minutes, you can start the game.
